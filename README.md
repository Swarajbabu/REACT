# 🚀 Learning MERN Stack

> A structured, hands-on journey through full-stack web development — from HTML basics to building complete web applications with the MERN stack.

---

## 📖 About This Repository

This repository documents a progressive learning path covering the core technologies of modern web development. Starting from the fundamentals of HTML and CSS, it advances through JavaScript, Node.js, Express, templating engines, and React — covering the complete skillset needed to build production-ready MERN (MongoDB, Express, React, Node.js) applications.

Whether you're a beginner looking for a structured roadmap or a developer revisiting concepts, this repo serves as a comprehensive reference and practice playground.

## ⚡ Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- A code editor like [VS Code](https://code.visualstudio.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Swarajbabu/LEARNING-MERN-STACK.git

# 2. Navigate into the project directory
cd LEARNING-MERN-STACK

# 3. Install dependencies
npm install
```

### Running Examples

Navigate into any topic folder and run the relevant file:

```bash
# Example: Run a Node.js script
cd 5-Nodejs
node app.js

# Example: Start an Express server
cd 6-EXPRESS
node index.js
```

## 📂 Highlights

### 📌 Projects (`0-PROJECTS/`)
Hands-on projects that combine multiple technologies learned throughout the journey. Each project reinforces real-world skills like building REST APIs, rendering dynamic views, and creating interactive UIs.

### ⚛️ React (`React/`)
Covers component architecture, JSX syntax, React hooks (`useState`, `useEffect`), props drilling, and building single-page applications.

### 🌐 Express.js (`6-EXPRESS/`)
Explores RESTful API design, route handling, middleware chains, and connecting the backend to templating engines.

### 🖥️ Node.js (`5-Nodejs/`)
Covers the Node.js event loop, built-in modules (`fs`, `path`, `http`), and npm package management.
